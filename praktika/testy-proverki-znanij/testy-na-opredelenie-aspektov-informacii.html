Тесты на определение аспекта информации

Теоретическая информация по теме "Аспекты"

Информационные аспекты. Субъективность восприятия

Семантика аспектов

 

ТЕСТ №1 для начинающих. Тест включает 20 словосочетаний с прямым значением слов. Например, "грустно до слез": значение - "энергетическое состояние", аспект ЧЭ.

ТЕСТ №2 для начинающих. Тест включает 20 словосочетаний с переносным значением слов. Аспект информации определяется по переносному значению. Например, "бить в точку": значение - "точно выявить суть", аспект ЧИ.

ТЕСТ №3  Тест включает 20 текстов, взятых из разных источников. Аспект информации определяется по смыслу всего текста. Для правильного определения аспекта, ответьте для себя на вопрос: "О чем этот текст?", сведя ответ к основным понятиям аспектов: об эмоциях, об отношениях, о соотношении в пространстве, о силе (границах), об объектах, о соотношении объектов, о возможностях (сути, смысле), о времени (соотношении возможностей). 

ТЕСТ №4  Тест включает 20 словосочетаний, которые могут иметь как прямое, так и переносное значение. Например, "вить веревки". Прямое значение - действие с объектом, аспект ЧЛ; переносное значение - подчинить своей воле, аспект ЧС. В данном тесте требуется определить аспект переносного значения словосочетания. 

ТЕСТ №5 Тест включает 20 текстов, взятых из разных источников. Аспект информации определяется по смыслу всего текста. Для правильного определения аспекта, ответьте для себя на вопрос: "О чем этот текст?", сведя ответ к основным понятиям аспектов: об эмоциях, об отношениях, о соотношении в пространстве, о силе (границах), об объектах, о соотношении объектов, о возможностях (сути, смысле), о времени (соотношении возможностей).
