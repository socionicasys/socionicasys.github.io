Требования к компьютеру:
 Установленная среда Java Runtime Version 6 Update 26 или выше.

Внимание! Для версии выше шестой необходимо изменить настройки безопасности Java согласно инструкции. Обратите внимание: в инструкции указан старый адрес сайта. Замените http://socionicasys.ru  на http://socionicasys.org

Также можете воспользоваться инструкцией, выложенной на форуме

Пошаговая инструкция установки Java:
Перейдите на сайт установки Java Runtime.
Укажите вашу операционную систему и загрузите Java Runtime Version 6 Update 26 или выше .
После загрузки установите Java Runtime согласно инструкции производителя.

Внимание! Эти тесты предназначены для обучения выявления параметров функций, но не для определения своего ТИМа. ШСС не использует тесты для определения ТИМа.
